When working on federal policy and advocacy, profiling data by state is fairly common, but creating a report for each state can be tedious. We found that an excellent way to go about this is to combine python and InDesign.

With a report template in mind and an idea of which data and text elements may vary by state, use python (pandas) to prepare these fields. They will be added from Excel via a mail merge. You can even test-run the report's text in python. But the best part is that with the font called FF Chartwell you can create high-quality charts from strings of numbers and plus signs. Fortunately, those fields are easy to prepare in pandas as well.

For the interactive code, download the Jupyter notebook and associated data files from GitHub.
