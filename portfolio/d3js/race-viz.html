These graphs were created as part of a project exploring the significant racial disparities that exist among people experiencing homelessness. For a bit more focus on D3, I've removed similar graphs and the rest of the project's paraphraphs, headers, subheaders, and navigation elements.

Some of the overall strategies implemented in this project include:

  • Responsiveness. Due to the amount of text, scaling a fixed aspect ratio was not ideal. Instead, the graphics have a fixed height that works for most phone browsers, and then the width is set depending on browser width. The SVG elements as well as text, dropdowns, etc. are placed so that they fit together at either any width (with some media queries involved as well). Resizing the window triggers a reload (try it!).

  • Scroll-based loading. For the sake of pretty entry animations, most visual aspects of each graphic are only loaded when the graphic comes into view, calculated based on the scrolling position of the element.

  • Subdued details. The "Explain This" and "Sources" pop-ups allow for the elimination of lengthy captions while allowing interested readers access to deeper information.
Switchable map, scatter, and bars
This chart allows you to view the same data in map, bar, or scatter plot form, each of which gives different insights into the data. Here's some of what's involved:
  • Clip paths on each
  • Zooming, panning, and re-setting
  • Various transitions
  • Icons and bezier curve
  • Selection retention between charts
  • The usual responsive axes, color scales, etc.
